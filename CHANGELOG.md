## [v1.6.0] - 2026-03-04

### ✨ 고도화 및 신규 기능 (Proton Balance)

- ⚖️ **NaOH 총량 분배 전략 및 목표 pH 최적화**
  - 고정 NaOH 모드 시 균등(Uniform), 전단집중(Front-loaded), 커스텀 비율 분배 지원
  - 단일 목표 pH 설정 시, 이분법(Bisection)을 사용하여 다단 시뮬레이션의 최종(Raffinate) pH가 목표치에 도달하도록 총 NaOH 자동 최적화
- 🧪 **Proton Balance 물질수지 정밀화**
  - 수계 버퍼 용량(Buffer Capacity) 모델 추가: 황산염(Sulfate) 해리 평형(pKa=1.99)에 기반하여 pH 변화 저항력 반영 (사이드바 황산염 농도 설정 추가)
  - 추출제 사포닌화(Saponification)에 의한 염기(OH⁻) 소모량을 물질수지에 반영하여 고pH 영역 계산 불일치 해결
  - 다단 고정 NaOH 모드에서 발생하는 pH 진동 발산을 개선하기 위한 Bisection Solver 도입
- 📉 **McCabe-Thiele 다이어그램 기능 추가**
  - 특정 금속에 대하여 Isotherm 평형 곡선(Equilibrium Curve)과 조작선(Operating Line), 역류 Stage 단수 프로파일을 직관적으로 시각화하는 탭 추가
- 📊 **차트 시각화 강화**
  - 단수별 pH 프로파일 및 NaOH 시각화에서 'Feed' 지점 추가. pH 증감을 한눈에 파악 가능

### 💡 1.6.0 업데이트 도입 효과

- **실제 조업과 유사한 완만한 pH 프로파일 모사**: 황산염 버퍼(Buffer Capacity)가 반영되어 급격한 pH Drop(곤두박질치는 현상) 방지
- **고(高) pH 영역 계산 정확도 비약적 향상**: 사포닌화 시 소모되는 염기(OH⁻) 양을 물질수지에 정밀하게 반영
- **목표 pH 도달 연산 최적화 및 안정성 확보**: Bisection Solver 도입으로 발산(Oscillation) 현상을 완벽히 해결하고, 빠르고 정확한 최적 총 NaOH 투입량 도출 가능

## [v1.4.0] - 2026-03-04

### ✨ 추가

- 🧪 **ESI 열역학 모델** 옵션 추가 (Lu et al., 2024)
  - `log(D) = log(C) + a × log([NaL])` 기반 분배계수 계산
  - 사포닌화 추출제 [NaL] 자동 계산 (K'a = 7.674×10⁻⁸)
  - 금속 간 추출제 경쟁이 열역학적으로 자동 반영
  - 사이드바에서 Sigmoid/ESI 모델 실시간 전환 가능
  - ⚠️ NaCl 시스템 기반 파라미터 (Sulfate 시스템 재피팅 필요)

### ✨ 추가

- ⚗️ **추출제 로딩 한계** 반영
  - 추출제 화학양론(n_ext) 파라미터 도입: 금속별 (HA)₂ dimer 소비량
  - 유기상 로딩률 계산: Σ(n_ext × C_M,org / MW) / C_ext
  - 시그모이드 감쇠 함수로 로딩 포화 시 추출 효율 자동 감소
  - 금속 간 추출제 경쟁 추출 반영 (반복 수렴)
  - 대시보드에 최대 로딩률 메트릭 표시 (85% 초과 시 경고)

## [v1.2.0] - 2026-03-04

### ✨ 추가

- 🌡️ **온도 의존성 모델** 추가
  - 반-경험적 온도 보정: pH₅₀(T) = pH₅₀(T_ref) + β·(T - T_ref)
  - 시그모이드 기울기 온도 보정: k(T) = k_ref · exp(γ·(T - T_ref))
  - 금속별 β, γ 온도 계수(사용자 조정 가능)
  - 대시보드 사이드바에 온도 슬라이더(10~60°C) 추가
  - 모델 수식 탭에 온도 보정 공식 표시
