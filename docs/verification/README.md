# SX 시뮬레이터 검증 문서

이 폴더에는 시뮬레이터의 검증 계획과 실행 결과를 기록합니다.

## 폴더 구조

```
docs/verification/
├── README.md              ← 이 파일
├── v1.7.0_plan.md         ← 검증 계획서
├── v1.7.0_results.md      ← 검증 실행 결과
└── (향후 버전별 추가)
```

## 규칙

1. **버전 단위로 문서 생성**: `v{버전}_plan.md` + `v{버전}_results.md`
2. **체크리스트 기반**: 모든 검증 항목은 `[ ]` / `[x]`로 추적
3. **수치 기록**: 테스트 결과의 핵심 수치(물질수지 오차, pH 오차 등)를 표로 기록
4. **이슈 발견 시**: 동일 문서에 "발견 이슈" 섹션으로 기록 후 수정 내용 추적
