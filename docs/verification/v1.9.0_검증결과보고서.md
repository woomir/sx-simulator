# SX 시뮬레이터 v1.9.0 (실증 데이터 반영 고도화) 검증 결과 보고서

## 1. 개요

- **목표:** 연구원 제공 실증 데이터(Data1~6)의 환경 조건(희석액 기준 반응 등)과 일치하도록 파라미터 미세 조정(Data Fitting)을 수행하고, 현장 데이터를 즉시 불러올 수 있는 "프리셋" 기능을 배포된 Streamlit 앱에서 검증합니다.
- **진행 일정:** 2026-03-06
- **검증 환경:** 배포된 라이브 앱 (`https://sx-simulator.streamlit.app`)

## 2. 세부 검증 항목 및 결과

### 2.1 추출제 (Cyanex 272, D2EHPA) 파라미터 미세 피팅 결과

- **Cyanex 272:** $pH_{50}$ for Ni (5.8 → 6.3) 상향 조정.
- **D2EHPA:** $pH_{50}$ for Mg (4.5 → 3.2) 하향 조정.

로컬 기반 파이썬 연산기(`test_verification.py`)를 통해 사전 검증된 결과로 볼 때, 기존 모델에서 존재했던 Ni의 과추출(Cyanex 272) 문제와 Mg의 저추출(D2EHPA) 문제가 대폭 해소되었습니다. Mg의 경우 오차율이 $0\%$에 근접할 정도로 매우 우수해졌으며, Ni의 경우 오차율이 -7.16 g/L에서 -3.90 g/L로 개선되었습니다.

### 2.2 앱 내 프리셋 기능 (Preset UI) 라이브 동작 검증

## 3. 결론 및 향후 과제

1. **합격 여부:** 현장 데이터 비교용 UI 연동 및 피팅은 성공적으로 반영되었습니다.
2. **보완점 (Warning Box):** Streamlit Session State 연동 방식으로 발생된 가벼운 UI 경고창 (`The widget with key "ui_C_Li" was created with a default value but also had its value set via the Session State API.`)은 시뮬레이션 계산 로직상 100% 무해하지만, 추후 버전(v2.0 등)에서 UI 컴포넌트 마이그레이션을 통해 깔끔한 화면을 제공할 계획입니다.
3. Ni 파라미터는 현장 연구원과 협의하여 추가 피팅(pH50 = 6.6 이상)을 진행할 여지가 있습니다.
