# v2.1.1 시뮬레이션 검증 누적 로그

이 문서는 시뮬레이션 엔진 변경 단계마다 수행한 검증 결과를 누적 기록한다.

검증 원칙:

- 구조 변경 단계라도 검증을 생략하지 않는다.
- 수치 결과가 의도적으로 변하지 않아야 하는 단계는 이전 패턴 유지 여부를 기록한다.
- 구조 검증과 현장 비교 검증을 분리해서 기록한다.

## 2단계 검증. 황산염 착물/자유 금속 speciation 연결

### 검증 목적

`K_MSO4`가 실제 speciation 계산에 연결되었는지 확인하고, 기존 시뮬레이션 결과를 불필요하게 깨지 않는지 확인한다.

### 수행 항목

1. 문법/구문 검증
2. speciation helper 직접 호출 확인
3. `validation_test.py`
4. `test_verification.py`

### 결과

#### 1. 문법/구문 검증

- `compile()` 기반 syntax check 통과
- 대상 파일 6개 모두 오류 없음

#### 2. speciation helper 직접 확인

조건:

- 금속: `Co`
- `pH = 5.5`
- `C_sulfate = 1.5 M`

결과:

- `free_metal_fraction = 0.002903`
- `hydroxo_fraction = 0.000000` 수준
- `sulfate_fraction = 0.997097`
- `free_sulfate_m = 1.499537`

해석:

- 고황산염 조건에서 Co는 대부분 sulfate complex 형태로 존재하도록 계산된다.
- 따라서 hydrolysis 기여가 매우 작아지고, `K_MSO4`가 실제 speciation 계산에 반영됨을 확인했다.

#### 3. `validation_test.py`

- Single stage mass balance: PASS
- Co isotherm consistency: PASS
- Multistage counter-current logic: PASS
- 다단 예제 수렴: `53 iterations`
- Co cascade balance diff: 약 `0.000050 g`

#### 4. `test_verification.py`

주요 후액 오차 패턴은 이전 단계와 동일하게 유지됐다.

- `Data1 Ni`: `-6.397 g/L`
- `Data2 Ni`: `-2.996 g/L`
- `Data4 Li`: `+2.160 g/L`
- `Data6 Co`: `-0.845 g/L`

금속별 MAE 요약도 유지:

- `Li`: `0.826 g/L`
- `Ni`: `1.943 g/L`
- `Co`: `0.206 g/L`
- `Mn`: `0.000 g/L`
- `Ca`: `0.000 g/L`
- `Mg`: `0.000 g/L`
- `Zn`: `0.000 g/L`

### 결론

2단계는 의도대로 `K_MSO4`를 실제 speciation 계산에 연결했고, 기존 현장 비교 패턴은 깨지지 않았다.

다만 이번 단계는 `pH/speciation submodel` 개선에 가깝기 때문에, 후액 농도 정확도 자체는 아직 크게 달라지지 않았다.

즉 다음 단계에서는 `free metal`을 추출 계산 쪽에 어떻게 연결할지 더 직접적으로 다뤄야 한다.
