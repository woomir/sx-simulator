## 6. 결론 및 권고

### 6.1 모델 구조 개선
- 4PL → 5PL Richards 함수 전환 성공 (하위 호환 유지)
- nu=1.0 금속은 4PL과 완전 동일 (차이 = 0)
- Li, Ca, Mg에 비대칭 파라미터(nu) 도입으로 곡선 표현력 향상

### 6.2 Li 모델링 개선
- Cyanex 272 Li: nu=1.8 (Fig.3b 기반)
- D2EHPA Li: nu=2.0 (Fig.2b 기반)
- 저pH에서 상승이 빠르고, 고pH에서 E_max에 완만하게 접근하는 비대칭 곡선 실현
- BM-CY.csv 현장 데이터에서 Li가 대부분 수계에 잔류하는 패턴과 일관

### 6.3 향후 과제
- BM-CY.csv/Ni_Cy.csv 다단 역류 데이터로 nu 파라미터 정밀 피팅
- 현장 데이터 기반 잔차 분석으로 체계적 편향 정량화
- 5PL 회귀 임계값 재보정
